Five Star Restaurant Reservations Booking Free Context

Provides the functionality for managing restaurant bookings from Five Star Restaurant Reservations â€“ WordPress Booking Plugin.

Additional notes:

  • Please be aware that this version of the context covers only the functionalities which are available form the free version.
  • In case the option "Allow Cancellations" is disabled, the Booking Cancellation will not work
  • In case the option "Require Phone" is set to "Yes", yuo will have to add the phone field into the payload for Create Appointment Element and Update Appointment Element

Currently this context can be used for the following functionalities:

  • Loading single booking for a user identified by email address
  • Loading multiple bookings for a user identified by email address
  • Checking availability of an booking
  • Suggesting available slots for an appointment type
  • Creating a new appointment for a user identified by email address
  • Updating an existing appointment for a user identified by email address
  • Canceling an existing appointment for a user identified by email address

Supported Payload keys and values set for Create Appointment Element and Update Appointment Element

  • message - additional message for the staff
  • name - customers full name under which the reservation will be recognized by the staff
  • party - number of people
  • phone - customers phone number on which the staff wil be able to contact the customer about the reservation

Parameters

  • Context ID - can be an expression like ${APPOINTMENT_TYPE} from the VARIABLES section or just a textual value
